## 🎮 How it works

The app asks:

👉 **"Do you love me?"**

* ✅ **Yes** → You win (with sound + popup 💖)
* ❌ **No** → The button escapes, shrinks, and trolls you 😂

---

## ✨ Features

* 🏃 Escaping button mechanics
* 🔊 Sound effects (Windows native — no extra libraries needed)
* 🔢 Live attempts counter
* 📊 Progress bar tracking button's remaining life
* 🎭 Random funny phrases
* 📉 Dynamic difficulty (button shrinks & darkens)
* 💀 Button disappears after 20 failed attempts
* 💬 Custom popup message based on your attempts count
* 🎨 Clean UI with custom icon
* 🖥️ Cross-platform support (Windows, Mac, Linux)

## 🚀 Run the App

### ▶️ From source

No extra libraries needed — just Python!

```bash
python love-btn.py
```

### 💻 From .exe

Download from **Releases** and run directly.

---

## 📦 Build EXE

```bash
pyinstaller --onefile --windowed --icon="assets/heart.ico" "love-btn.py"
```

The `.exe` will be in the `dist/` folder.

---

## 📁 Assets

Only the icon is needed now:

```
assets/
 └── heart.ico
```

> 🔊 Sound effects are handled by the OS — no `.wav` or `.mp3` files required.
